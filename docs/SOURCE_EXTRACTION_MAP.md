# Mapa de extracción de Genesis-core y Genesis-corev2

## Conservar de Genesis-core

- separación entre semilla, runtime, memoria, motor y guardián;
- neutralidad de plataforma y proveedor;
- procedencia y estados epistémicos;
- replay determinista;
- privacidad append-only;
- aprobación gobernada;
- crecimiento con puertas;
- portabilidad y versionado por capas.

## Conservar de Genesis-corev2

- protocolo explícito;
- nacimiento transaccional;
- manifiesto de semilla;
- canonicalización especificada;
- vectores dorados;
- cadena de memoria;
- verificación anclada;
- implementaciones de referencia;
- niveles de conformidad.

## No convertir en norma

- Kotlin/JVM;
- Node.js;
- Python;
- Rego/OPA;
- Android, Apple o Windows;
- una base de datos concreta;
- un modelo de IA específico.

## Nuevos elementos

- identidad de instancia separada del dispositivo;
- registro de cuerpos;
- transferencia normal;
- recuperación de emergencia;
- revocación de cuerpos perdidos;
- declaración de brechas de memoria;
- prevención de bifurcaciones;
- paquetes de transferencia independientes del transporte.
